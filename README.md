# ગુજરાતી વ્યાકરણ 📚

ગુજરાતી વ્યાકરણ શીખવા માટેનું Web App.

## Features
- 📖 **થિયરી** - 39 પ્રકરણોની સંપૂર્ણ થિયરી
- 📝 **ટેસ્ટ** - દરેક પ્રકરણની MCQ ટેસ્ટ
- 📄 **PDF** - Theory & Questions PDF viewer
- 📱 **PWA** - Install as mobile app
- 🔤 **Font Size** - Adjustable text size

## Tech Stack
- React 19 + Vite 7
- React Router DOM
- React PDF
- Vercel Deployment
